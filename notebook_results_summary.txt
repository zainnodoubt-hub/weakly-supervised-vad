# Weakly Supervised Video Anomaly Detection (MIL) — Frame-based Pipeline

## Dataset format

The dataset consists of **extracted PNG frames**, not raw videos. Example paths:

- **Local (default):** `datasets/ucf-crime-dataset/Train/Arson/Arson042_x264_1170.png`
- **Local (default):** `datasets/ucf-crime-dataset/Train/NormalVideos/Normal_Videos006_x264_10.png`
- **Colab:** e.g. `/content/drive/MyDrive/.../UCF_Crime/...` if `USE_GOOGLE_DRIVE = True` in Section 2.

**Clip definition:** Files with the same prefix before the final `_<number>` belong to one clip.

- `Arson042_x264_1170.png` and `Arson042_x264_1560.png` → **clip_id** = `Arson042_x264`
- **Frame index** = the number after the final underscore.

**Labels:** `NormalVideos` folder → label **0**; all other class folders → label **1**.

## Why frames instead of videos

- Frames are already extracted; no need for OpenCV video decoding.
- Faster and more reliable than decoding full videos from cloud or network drives.
- Same MIL pipeline: clips → segments → features → training.

## Pipeline overview

This notebook supports **UCF-Crime** (frame-based) and **XD-Violence** (I3D features). Config (Section 2) controls which dataset(s) are used.

0. *(Local)* Install deps (Section 0) if needed → 1. Config (paths) → 2. Build/load frame metadata → 3. Clip-level splits → 4. Segment sampling → 5. ResNet feature extraction → 6. Save feature bags → 7. Export manifests for MIL. *(On Colab only, run Drive mount in Section 1 first.)*

def detect_hardware():
    """Detect available hardware and return the appropriate PyTorch variant"""
    system = platform.system()
    machine = platform.machine()
    
    print("=" * 60)
    print("SYSTEM DETECTION")
    print("=" * 60)
    print(f"Operating System: {system}")
    print(f"Architecture: {machine}")
    
    # Check for NVIDIA GPU
    has_nvidia = False
    nvidia_gpu_name = None
    cuda_driver_version = None
    
    try:
        result = subprocess.run(['nvidia-smi', '--query-gpu=name', '--format=csv,noheader'], 
                               capture_output=True, text=True, timeout=5)
        if result.returncode == 0 and result.stdout.strip():
            has_nvidia = True
            nvidia_gpu_name = result.stdout.strip().split('\n')[0]
            
            result2 = subprocess.run(['nvidia-smi', '--query-gpu=driver_version', '--format=csv,noheader'],
                                    capture_output=True, text=True, timeout=5)
            if result2.returncode == 0:
                cuda_driver_version = result2.stdout.strip().split('\n')[0]
            
            print(f"✓ NVIDIA GPU detected: {nvidia_gpu_name}")
            print(f"✓ CUDA Driver: {cuda_driver_version}")
    except:
        pass
    
    # Check for Apple Silicon
    is_apple_silicon = (system == 'Darwin' and machine == 'arm64')
    if is_apple_silicon:
        print("✓ Apple Silicon (M1/M2/M3) detected")
    
    print("-" * 60)
    
    return {
        'has_nvidia': has_nvidia,
        'nvidia_gpu': nvidia_gpu_name,
        'cuda_driver': cuda_driver_version,
        'is_apple_silicon': is_apple_silicon,
        'system': system,
        'machine': machine
    }


def get_pytorch_install_cmd(hardware):
    """Return the appropriate PyTorch install command"""
    
    if hardware['has_nvidia']:
        cuda_driver = hardware['cuda_driver']
        cuda_version = "cu118"
        
        if cuda_driver:
            driver_major = int(cuda_driver.split('.')[0])
            if driver_major >= 12:
                cuda_version = "cu121"
                print(f"✓ Using CUDA 12.1 (driver {cuda_driver})")
            else:
                print(f"✓ Using CUDA 11.8 (driver {cuda_driver})")
        
        print(f"Installing PyTorch with CUDA {cuda_version}...")
        return [sys.executable, "-m", "pip", "install", "--upgrade",
                "torch", "torchvision", "torchaudio",
                "--index-url", f"https://download.pytorch.org/whl/{cuda_version}"]
    
    elif hardware['is_apple_silicon']:
        print("Installing PyTorch with MPS support...")
        return [sys.executable, "-m", "pip", "install", "--upgrade",
                "torch", "torchvision", "torchaudio"]
    
    else:
        print("Installing PyTorch CPU version...")
        return [sys.executable, "-m", "pip", "install", "--upgrade",
                "torch", "torchvision", "torchaudio",
                "--index-url", "https://download.pytorch.org/whl/cpu"]

def build_xd_metadata(i3d_dir: Path) -> pd.DataFrame:
    """Scan I3D dir recursively for .npy; split from parent folder (Flow/RGB=train, FlowTest/RGBTest=test); label from filename (label_A=0)."""
    i3d_dir = Path(i3d_dir)
    rows = []
    for p in tqdm(list(i3d_dir.rglob("*.npy")), desc="XD scan"):
        try:
            rel = p.relative_to(i3d_dir)
            parent = rel.parts[0] if rel.parts else p.parent.name
            # Train = Flow, RGB; Test = FlowTest, RGBTest
            split = "train" if parent in ("Flow", "RGB") else "test"
            # label_A in filename -> normal (0), else violence (1)
            label = 0 if "label_A" in p.name else 1
            clip_id = str(rel).replace("/", "__")
            rows.append({
                "clip_id": clip_id,
                "feature_path": str(p.resolve()),
                "label": label,
                "split": split,
                "dataset": "xd",
            })
        except Exception as e:
            continue
    return pd.DataFrame(rows)

# Build XD feature bags: subsample I3D to NUM_SEGMENTS, project to FEATURE_DIM, save to FEATURES_DIR_XD.
# Run after Phase 2 (xd_train_df, xd_val_df exist). Cap for minimal data; set None for full.
import numpy as np
from pathlib import Path
from tqdm import tqdm

XD_FEATURE_CAP_PER_CLASS = None  # None = all; 4 = minimal for E2E test

def subsample_and_project_xd_bag(raw: np.ndarray, num_segments: int, out_dim: int, in_dim: int) -> np.ndarray:
    """raw shape (T_raw, in_dim) -> (num_segments, out_dim). Subsample with linspace; pad in_dim to out_dim."""
    T = raw.shape[0]
    if T == 0:
        return np.zeros((num_segments, out_dim), dtype=np.float32)
    idx = np.linspace(0, T - 1, num_segments, dtype=int)
    segs = raw[idx]  # (num_segments, in_dim)
    if out_dim > in_dim:
        segs = np.concatenate([segs, np.zeros((num_segments, out_dim - in_dim), dtype=np.float32)], axis=1)
    elif out_dim < in_dim:
        segs = segs[:, :out_dim]
    return segs.astype(np.float32)

def build_xd_bags_for_df(df, out_dir: Path, cap_per_class: int, overwrite: bool = False) -> int:
    out_dir = Path(out_dir)
    out_dir.mkdir(parents=True, exist_ok=True)
    if len(df) == 0:
        return 0
    subset = df
    if cap_per_class is not None:
        parts = [df[df["label"] == lab].head(cap_per_class) for lab in df["label"].unique()]
        subset = pd.concat(parts, ignore_index=True)
    n_saved = 0
    for _, row in tqdm(subset.iterrows(), total=len(subset), desc="XD bags"):
        clip_path = Path(row["clip_id"])  # e.g., Flow/Salt.2010__...npy
        out_path = out_dir / clip_path    # preserve folder structure
        
        # 🔥 CRITICAL: Create subdirectories (Flow/, RGB/, etc.)
        out_path.parent.mkdir(parents=True, exist_ok=True)
        
        if out_path.exists() and not overwrite:
            n_saved += 1
            continue
        try:
            raw = np.load(row["feature_path"]).astype(np.float32)
            bag = subsample_and_project_xd_bag(raw, NUM_SEGMENTS, FEATURE_DIM, XD_FEATURE_DIM)
            np.save(out_path, bag)
            n_saved += 1
        except Exception as e:
            # Better debugging - don't silently fail
            print(f"Error saving {out_path}: {e}")
            continue
    return n_saved

if "xd_train_df" in dir() and "xd_val_df" in dir() and DATASET_MODE in ("xd", "both"):

import pandas as pd

def stratified_clip_split(df: pd.DataFrame, train_ratio: float, val_ratio: float, test_ratio: float, seed: int) -> tuple:
    if len(df) == 0:
        cols = [c for c in ["dataset", "clip_id", "split", "class_name", "label", "num_frames", "frame_paths"] if c in df.columns]
        if not cols:
            cols = ["dataset", "clip_id", "split", "class_name", "label", "num_frames", "frame_paths"]
        return pd.DataFrame(columns=cols), pd.DataFrame(columns=cols), pd.DataFrame(columns=cols)
    train_parts, val_parts, test_parts = [], [], []
    for (dataset, label), group in df.groupby(["dataset", "label"]):
        group = group.sample(frac=1.0, random_state=seed).reset_index(drop=True)
        n = len(group)
        n_train = max(1, int(round(n * train_ratio)))
        n_val = max(1, int(round(n * val_ratio)))
        n_test = max(1, n - n_train - n_val)
        while n_train + n_val + n_test > n:
            n_train = max(1, n_train - 1)
        train_parts.append(group.iloc[:n_train])
        val_parts.append(group.iloc[n_train:n_train + n_val])
        test_parts.append(group.iloc[n_train + n_val:n_train + n_val + n_test])
    train_df = pd.concat(train_parts, ignore_index=True).sample(frac=1.0, random_state=seed).reset_index(drop=True)
    val_df = pd.concat(val_parts, ignore_index=True).sample(frac=1.0, random_state=seed).reset_index(drop=True)
    test_df = pd.concat(test_parts, ignore_index=True).sample(frac=1.0, random_state=seed).reset_index(drop=True)
    return train_df, val_df, test_df

## 6. Segment sampling

**Strategy:** each clip is split into **uniform segments** along the frame index (`np.linspace` boundaries), not a sliding window. Within each segment, frames are sampled at **uniform indices** for the backbone.

`segment_boundaries(num_frames, num_segments)` and `load_segment_frames_from_list(frame_paths, start_idx, end_idx, ...)` return frames as `[T, H, W, C]`.

# Define segment_boundaries() and load_segment_frames_from_list(); used to sample frames per segment for feature extraction.
import numpy as np
import cv2
from pathlib import Path
from typing import List, Tuple
from PIL import Image

def segment_boundaries(num_frames: int, num_segments: int) -> List[Tuple[int, int]]:
    if num_frames <= 0:
        return [(0, 0)] * num_segments
    edges = np.linspace(0, num_frames, num_segments + 1, dtype=int)
    return [(int(edges[i]), int(max(edges[i + 1], edges[i] + 1))) for i in range(num_segments)]

def sample_indices(start: int, end: int, num_samples: int) -> List[int]:
    if end <= start:
        return [start] * num_samples
    idxs = np.linspace(start, end - 1, num_samples)
    return [int(x) for x in idxs]

def load_frames(frame_paths: List[str]) -> np.ndarray:
    frames = []
    for p in frame_paths:
        img = Image.open(p).convert("RGB")
        frames.append(np.array(img))
    return np.stack(frames, axis=0)

def load_segment_frames_from_list(
    frame_paths: List[str],
    start_idx: int,
    end_idx: int,
    num_samples: int = 16,
    resize_hw: Tuple[int, int] = (224, 224),
) -> np.ndarray:
    n = len(frame_paths)
    if n == 0:
        return np.zeros((num_samples, resize_hw[0], resize_hw[1], 3), dtype=np.uint8)
    indices = sample_indices(start_idx, min(end_idx, n), num_samples)
    paths = [frame_paths[min(i, n - 1)] for i in indices]
    frames = load_frames(paths)
    out = np.stack([cv2.resize(f, (resize_hw[1], resize_hw[0])) for f in frames], axis=0)
    return out.astype(np.uint8)

# --- R(2+1)D clip extraction (in-notebook; no external .py). Used when USE_3D_FEATURES=True. ---
# Stacked ordered PNGs -> Kinetics-pretrained R(2+1)D-18 -> bag [num_clips, 512].
from pathlib import Path
from typing import List, Sequence

import numpy as np
import torch
import torch.nn as nn
from PIL import Image
from torchvision.models.video import R2Plus1D_18_Weights, r2plus1d_18


def sample_clip_paths(frame_paths: Sequence[str], clip_len: int, stride: int) -> List[List[str]]:
    """Overlapping clips of frame paths. Short videos: pad by repeating the last frame."""
    paths = list(frame_paths)
    n = len(paths)
    if n == 0:
        return []
    if n < clip_len:
        pad = [paths[-1]] * (clip_len - n)
        return [paths + pad]
    clips: List[List[str]] = []
    for start in range(0, n - clip_len + 1, stride):
        clips.append(paths[start : start + clip_len])
    return clips


def load_clip_tensor_tchw(frame_paths: Sequence[str]) -> torch.Tensor:
    """Load PNG paths to uint8 tensor [T, C, H, W] for R2Plus1D_18_Weights.transforms()."""
    frames = []
    for p in frame_paths:
        img = Image.open(p).convert("RGB")
        frames.append(np.asarray(img, dtype=np.uint8))
    vid = np.stack(frames, axis=0)
    return torch.from_numpy(vid).permute(0, 3, 1, 2).contiguous()


class R2Plus1DClipExtractor(nn.Module):
    """One clip -> 512-d vector (Kinetics-pretrained R(2+1)D-18, global pool)."""

    def __init__(self, device=None):
        super().__init__()
        weights = R2Plus1D_18_Weights.DEFAULT
        self.preprocess = weights.transforms()
        self.backbone = r2plus1d_18(weights=weights)
        self.output_dim = 512
        self._device = device or torch.device("cpu")
        self.to(self._device)

    @torch.no_grad()
    def forward(self, vid_tchw: torch.Tensor) -> torch.Tensor:
        vid_tchw = vid_tchw.cpu()
        x = self.preprocess(vid_tchw)
        if x.dim() == 4:
            x = x.unsqueeze(0)
        x = x.to(self._device)
        x = self.backbone.stem(x)
        x = self.backbone.layer1(x)
        x = self.backbone.layer2(x)
        x = self.backbone.layer3(x)
        x = self.backbone.layer4(x)
        x = self.backbone.avgpool(x)
        return x.flatten(1).squeeze(0)


def extract_r2plus1d_bag(
    frame_paths: Sequence[str],
    clip_len: int,
    stride: int,
    extractor: R2Plus1DClipExtractor,
) -> np.ndarray:
    """Return array [num_clips, 512]."""
    rows = []
    for cp in sample_clip_paths(frame_paths, clip_len, stride):
        vid = load_clip_tensor_tchw(cp)
        feat = extractor(vid).float().cpu().numpy()
        rows.append(feat)
    if not rows:
        return np.zeros((0, extractor.output_dim), dtype=np.float32)
    return np.stack(rows, axis=0).astype(np.float32)


def extract_and_save_r2plus1d_for_df(df, out_dir: Path, device: torch.device, clip_len: int, stride: int, overwrite: bool = False):
    """One .npy bag per clip_id under out_dir (same role as extract_and_save_clip_features)."""
    import pandas as pd
    from tqdm import tqdm

    out_dir = Path(out_dir)
    out_dir.mkdir(parents=True, exist_ok=True)
    extractor = R2Plus1DClipExtractor(device=device).eval()
    rows = []
    for _, row in tqdm(df.iterrows(), total=len(df), desc="R(2+1)D clips"):
        clip_id = row["clip_id"]
        frame_paths = list(row["frame_paths"])
        bag_path = out_dir / f"{clip_id}.npy"
        if bag_path.exists() and not overwrite:
            bag = np.load(bag_path)
        else:
            bag = extract_r2plus1d_bag(frame_paths, clip_len, stride, extractor)
            np.save(bag_path, bag)
        label = int(row["label"])
        rows.append(
            {
                "clip_id": clip_id,
                "label": label,
                "feature_path": str(bag_path),
                "num_segments": bag.shape[0],
                "feature_dim": bag.shape[1],
            }
        )
    return pd.DataFrame(rows)

# PyTorch Dataset: loads .npy feature bags; optional norm_stats (mean, std) applied (fit on train, same for val/test).
import torch
from torch.utils.data import Dataset
import numpy as np

class MILFeatureDataset(Dataset):
    def __init__(self, manifest_df, norm_stats=None):
        self.df = manifest_df.reset_index(drop=True)
        self.norm_stats = norm_stats  # (mean, std) each shape (D,) or None

    def __len__(self):
        return len(self.df)

    def __getitem__(self, idx):
        row = self.df.iloc[idx]
        features = np.load(row.feature_path).astype(np.float32)
        if self.norm_stats is not None:
            mean, std = self.norm_stats
            features = (features - mean) / std
        T = features.shape[0]
        return torch.tensor(features, dtype=torch.float32), torch.tensor(row.label), torch.tensor(T, dtype=torch.long)

from torch.utils.data import DataLoader, Sampler

class BalancedBatchSampler(Sampler):
    """Yield batch indices so each batch has (approx) equal normal and anomaly samples."""
    def __init__(self, labels, batch_size, seed=42):
        self.labels = np.asarray(labels)
        self.batch_size = batch_size
        self.seed = seed
        self.normal_idx = np.where(self.labels == 0)[0]
        self.anomaly_idx = np.where(self.labels == 1)[0]

    def __iter__(self):
        rng = np.random.default_rng(self.seed)
        n_half = self.batch_size // 2
        n_batches = (len(self.labels) + self.batch_size - 1) // self.batch_size
        for _ in range(n_batches):
            n_n = min(n_half, len(self.normal_idx))
            n_a = min(self.batch_size - n_n, len(self.anomaly_idx))
            if n_n == 0: n_n = min(1, len(self.normal_idx))
            if n_a == 0: n_a = min(1, len(self.anomaly_idx))
            idx_n = rng.choice(self.normal_idx, size=n_n, replace=len(self.normal_idx) < n_n)
            idx_a = rng.choice(self.anomaly_idx, size=n_a, replace=len(self.anomaly_idx) < n_a)
            yield list(np.concatenate([idx_n, idx_a]))

    def __len__(self):
        return (len(self.labels) + self.batch_size - 1) // self.batch_size

def collate_pad_bags(batch):
    """Pad variable-length feature bags to max T in batch; return (feats, labels, lengths)."""

    labels = torch.stack([b[1] for b in batch])
    lengths = torch.stack([b[2] for b in batch])
    return feats, labels, lengths

# Temporal self-attention over segments (required improvement). Set USE_ATTENTION_MODEL = True to use.
# Custom layer so we can return attention weights for heatmap visualisation.
class TransformerEncoderLayerWithWeights(nn.Module):
    def __init__(self, d_model, nhead, dim_feedforward, dropout=0.1):
        super().__init__()
        self.self_attn = nn.MultiheadAttention(d_model, nhead, dropout=dropout, batch_first=True)
        self.linear1 = nn.Linear(d_model, dim_feedforward)
        self.linear2 = nn.Linear(dim_feedforward, d_model)
        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        self.dropout = nn.Dropout(dropout)

    def forward(self, src):
        src2, attn_weights = self.self_attn(src, src, src, need_weights=True)  # attn_weights: [B, T, T]
        src = src + self.dropout(src2)
        src = self.norm1(src)
        src2 = self.linear2(self.dropout(torch.nn.functional.relu(self.linear1(src))))
        src = src + self.dropout(src2)
        src = self.norm2(src)
        return src, attn_weights

class MILModelWithAttention(nn.Module):
    def __init__(self, feature_dim=2048, hidden_dim=256, num_heads=4, dropout=0.3, input_dim=None):
        super().__init__()
        input_dim = input_dim if input_dim is not None else feature_dim
        self.input_proj = nn.Linear(input_dim, feature_dim) if input_dim != feature_dim else nn.Identity()
        self.proj = nn.Linear(feature_dim, hidden_dim)
        self.layer1 = TransformerEncoderLayerWithWeights(hidden_dim, num_heads, hidden_dim * 4, dropout)
        self.layer2 = TransformerEncoderLayerWithWeights(hidden_dim, num_heads, hidden_dim * 4, dropout)
        self.scorer = nn.Sequential(
            nn.Linear(hidden_dim, 128),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(128, 1),
        )
        self.return_attention = False  # Set True when you need weights for heatmap

    def forward(self, x):
        x = self.input_proj(x)
        x = self.proj(x)
        x, _ = self.layer1(x)
        x, attn_weights = self.layer2(x)  # [B, T, T]
        logits = self.scorer(x).squeeze(-1)
        if self.return_attention:
            return logits, attn_weights
        return logits

# Ranking loss (Sultani-style): ranking + smoothness over segments + sparsity. Optional top-K for video score.
import torch

def ranking_loss(anomaly_scores, normal_scores, margin=1.0):
    """Anomaly scores should be higher than normal. Works when batch sizes differ."""
    max_anom = anomaly_scores.max(dim=1)[0]
    max_norm = normal_scores.max(dim=1)[0]
    diff = max_anom.unsqueeze(1) - max_norm.unsqueeze(0)
    loss = torch.relu(margin - diff)
    return loss.mean()

def ranking_loss_sultani(anomaly_scores, normal_scores, margin=1.0, lambda_smooth=8e-5, lambda_sparse=8e-5):
    """Full Sultani et al.: ranking + temporal smoothness (on anomaly) + sparsity (on anomaly)."""
    rank = ranking_loss(anomaly_scores, normal_scores, margin=margin)
    smooth = 0.0
    if anomaly_scores.shape[1] > 1:
        smooth = ((anomaly_scores[:, 1:] - anomaly_scores[:, :-1]) ** 2).mean()
    sparse = anomaly_scores.mean()
    return rank + lambda_smooth * smooth + lambda_sparse * sparse

def topk_mean(scores, k=None):
    """Video-level score: mean of top-k segment scores. scores: [B, T]. k=None -> max(1, T//8)."""
    T = scores.shape[1]
    if k is None:
        k = max(1, T // 8)
    k = min(k, T)
    topk_vals = torch.topk(scores, k, dim=1).values
    return topk_vals.mean(dim=1)

def topk_mean_masked(scores, lengths, k=None):
    """Video-level score over valid segments only. scores: [B, T], lengths: [B]."""
    B = scores.shape[0]
    out = []
    for i in range(B):
        t = lengths[i].item()
        if t <= 0:
            out.append(0.0)
            continue
        segs = scores[i, :t]
        kk = min(k or max(1, t // 8), t)
        out.append(segs.topk(kk).values.mean().item())
    return torch.tensor(out, device=scores.device, dtype=scores.dtype)

def ranking_loss_sultani_masked(anomaly_scores, normal_scores, len_anom, len_norm, margin=1.0, lambda_smooth=8e-5, lambda_sparse=8e-5):
    """Sultani loss with variable-length bags: mask padded positions."""
    T = anomaly_scores.shape[1]
    mask_anom = (torch.arange(T, device=anomaly_scores.device).unsqueeze(0) < len_anom.unsqueeze(1))
    mask_norm = (torch.arange(T, device=normal_scores.device).unsqueeze(0) < len_norm.unsqueeze(1))
    a = anomaly_scores.clone(); a[~mask_anom] = -1e9
    n = normal_scores.clone(); n[~mask_norm] = -1e9
    rank = ranking_loss(a, n, margin=margin)
    smooth = 0.0
    for i in range(anomaly_scores.shape[0]):
        L = len_anom[i].item()
        if L > 1:
            smooth = smooth + ((anomaly_scores[i, 1:L] - anomaly_scores[i, :L-1]) ** 2).mean()
    if anomaly_scores.shape[0] > 0:
        smooth = smooth / anomaly_scores.shape[0]
    sparse = 0.0
    cnt = 0
    for i in range(anomaly_scores.shape[0]):
        L = len_anom[i].item()
        if L > 0:
            sparse = sparse + anomaly_scores[i, :L].mean(); cnt += 1
    if cnt > 0:
        sparse = sparse / cnt
    return rank + lambda_smooth * smooth + lambda_sparse * sparse

import numpy as np
import torch
from sklearn.metrics import average_precision_score, roc_auc_score


def max_pool_logits_masked(logits: torch.Tensor, lengths: torch.Tensor) -> torch.Tensor:
    """Max over valid segment logits per video."""
    B = logits.shape[0]
    out = []
    for i in range(B):
        t = int(lengths[i].item())
        if t <= 0:
            out.append(0.0)
        else:
            out.append(logits[i, :t].max().item())
    return torch.tensor(out, device=logits.device, dtype=logits.dtype)


def collect_video_scores_mil(model, loader, device, pooling: str, topk_mean_masked_fn):
    """pooling: 'topk' matches val; 'max' = max logits then sigmoid."""
    if pooling not in ("topk", "max"):
        raise ValueError(pooling)
    model.eval()
    scores_all, labels_all = [], []
    with torch.no_grad():
        for features, labels, lengths in loader:
            features = features.to(device)
            lengths = lengths.to(device)
            out = model(features)
            logits = out[0] if isinstance(out, tuple) else out
            if pooling == "topk":
                vs = torch.sigmoid(topk_mean_masked_fn(logits, lengths)).cpu().numpy()
            else:
                vs = torch.sigmoid(max_pool_logits_masked(logits, lengths)).cpu().numpy()
            scores_all.extend(vs.tolist())
            labels_all.extend(labels.numpy().tolist())
    return np.asarray(scores_all, dtype=np.float64), np.asarray(labels_all, dtype=np.float64)


def video_auc_ap(labels: np.ndarray, scores: np.ndarray):
    if len(labels) == 0 or len(np.unique(labels)) < 2:
        return float("nan"), float("nan")
    return (
        float(roc_auc_score(labels, scores)),
        float(average_precision_score(labels, scores)),
    )


def segment_expanded_roc_auc_approx(manifest_df, video_scores: np.ndarray, video_labels: np.ndarray) -> float:
    """Segment-expanded ROC AUC — APPROXIMATE (not true frame-level GT)."""
    frame_labels, frame_scores = [], []
    for i in range(len(manifest_df)):
        row = manifest_df.iloc[i]
        bag = np.load(row.feature_path)
        t = int(bag.shape[0])
        lab = int(video_labels[i]) if i < len(video_labels) else int(row.label)
        vid_score = float(video_scores[i]) if i < len(video_scores) else 0.0
        for _ in range(t):
            frame_labels.append(lab)
            frame_scores.append(vid_score)
    if len(frame_labels) == 0 or len(set(frame_labels)) < 2:
        return float("nan")
    return float(roc_auc_score(frame_labels, frame_scores))

### Improvement narrative and terminology

**Baseline MIL weakness:** Standard MIL aggregates segment scores (e.g. max or mean) without modelling temporal context. The model sees each segment largely in isolation, so it can be sensitive to a single anomalous segment (max) or diluted by many normal segments (mean). There is no explicit mechanism to weigh which segments are most relevant.

**Why temporal self-attention helps:** The attention-based model applies *temporal self-attention over segment features*: after the shared MLP, a transformer encoder lets each segment attend over all segments. So the model can use context (e.g. surrounding segments) and can assign higher weight to more discriminative segments. This is **temporal** self-attention over *segment-level* representations only; we do **not** use spatial attention over raw frames, so we describe it as temporal self-attention over segment features, not “spatiotemporal” attention.

**What the experiments show:** The comparison table below trains baseline MIL and attention MIL under the same conditions (data, loss, hyperparameters) and reports Video AUC (and AP). We expect the attention model to match or improve over the baseline when the improvement is real. We also compare aggregation (max vs Top-K) to show that Top-K pooling can improve stability.

**Where the improvement still fails:** Weak supervision (video-level labels only) remains a limit: we do not have frame-level labels, so segment-expanded “frame” ROC is an approximation. The model can still produce false positives (e.g. unusual but normal motion) and false negatives (e.g. subtle or brief anomalies), and the failure analysis section discusses these cases.

            out = model(features)

            n_mask, a_mask = labels == 0, labels == 1
            sn, sa = scores[n_mask], scores[a_mask]
            ln, la = lengths[n_mask], lengths[a_mask]
            if len(sn) > 0 and len(sa) > 0:
                loss = ranking_loss_sultani_masked(sa, sn, la, ln, margin=1.0, lambda_smooth=8e-5, lambda_sparse=8e-5)
                epoch_batch_losses.append(loss.item())  # ← FIX: ADD THIS
                optimizer.zero_grad()
                loss.backward()
                torch.nn.utils.clip_grad_norm_(model.parameters(), 1.0)
                optimizer.step()
            else:

                    if labels[i].item() == 0: acc_n.append((features[i:i+1], lengths[i:i+1]))

                while len(acc_n) >= 1 and len(acc_a) >= 1:
                    n_take = min(len(acc_n), len(acc_a), 8)
                    fn = torch.cat([acc_n[j][0] for j in range(n_take)], dim=0)
                    fa = torch.cat([acc_a[j][0] for j in range(n_take)], dim=0)
                    ln_ = torch.cat([acc_n[j][1] for j in range(n_take)], dim=0)
                    la_ = torch.cat([acc_a[j][1] for j in range(n_take)], dim=0)
                    acc_n[:n_take], acc_a[:n_take] = [], []
                    T = max(fn.shape[1], fa.shape[1])
                    if fn.shape[1] < T: fn = F.pad(fn, (0,0,0,T-fn.shape[1]))

                    epoch_batch_losses.append(loss.item())
                    optimizer.zero_grad(); loss.backward(); torch.nn.utils.clip_grad_norm_(model.parameters(), 1.0); optimizer.step()
        if epoch_batch_losses:
            train_losses.append(float(np.mean(epoch_batch_losses)))
        model.eval()
        val_scores, val_labels = [], []
        with torch.no_grad():
            for feats, lbls, lens in val_loader:
                feats, lens = feats.to(device), lens.to(device)
                out = model(feats); s = out[0] if isinstance(out, tuple) else out
                vs = torch.sigmoid(topk_mean_masked(s, lens)).cpu().numpy()
                val_scores.extend(vs); val_labels.extend(lbls.numpy())
        if len(val_scores) > 0 and len(set(val_labels)) > 1:
            auc = roc_auc_score(val_labels, val_scores)
            ap = average_precision_score(val_labels, val_scores)

# --- TOP-K POOLING: Why it improves MIL stability ---

Standard MIL uses **max pooling** (single highest segment score) to get video-level score. This is sensitive to outliers and ignores temporal context.

**Top-K pooling** averages the K highest segment scores per video.

**Benefits:**
1. More robust to noisy single-segment predictions
2. Captures anomalies that span multiple segments
3. Provides gradient signal to multiple segments, not just the max
4. Smoother training and better generalization

====================================================================================================
CELL 51 | TYPE: MARKDOWN
====================================================================================================

[MARKDOWN]
**Why Top-K pooling can improve MIL stability:** In MIL we aggregate segment scores to a single video score. **Max pooling** uses only the single highest-scoring segment; it is sensitive to one noisy or outlier segment and ignores the rest. **Top-K mean pooling** (e.g. mean over the top-K segment scores per video) uses several highly scoring segments, so the video score is more stable and less driven by a single segment. This can improve robustness and often gives better validation AUC when anomalies span multiple segments or when max is overly sensitive.

### Failure analysis

**Definitions:** A **false positive (FP)** is a video that is normal (label 0) but the model predicts a high anomaly score (e.g. above the chosen threshold). A **false negative (FN)** is an anomalous video (label 1) that receives a low score. The failure-case bank above lists example clip IDs for FPs and FNs.

**Possible causes:**

- **Context bias:** The model may rely on scene or context cues that correlate with the label in training but not in reality (e.g. a particular location or time of day), leading to FPs when normal videos share that context.
- **Unusual but normal motion:** Normal videos with fast motion, crowds, or rare activities can get high scores because they look “different” from the typical normal pattern, causing FPs.
- **Weak supervision ambiguity:** With only video-level labels, we do not know which segments are truly anomalous. The model may attend to or score the wrong parts of the video, or miss brief anomalies, leading to both FPs and FNs.
- **Short or subtle anomalies:** Brief or visually subtle anomalies may not change the aggregated video score enough, causing FNs.

Interpreting the failure bank and the anomaly score curves (e.g. the third plot) helps identify where the method still fails and guides future improvements (e.g. better features or segment-level constraints).
